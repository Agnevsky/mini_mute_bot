# mini_mute_bot
Простой тг бот для мьюта юзера на определенное(от 1 минуты до 60 минут) время в супергруппе

Работает абсолютно просто, получает id пользователя на сообщение которого ответили и мьютит его на какое-то время
Достаточно в чате, в ответ на сообщение, написать !mute <time> (пример: !mute 20)
Время указывается в минутах
